## Linting and Formatting

Provided by [Prettier](https://prettier.io/) and [ESLint](https://eslint.org/).

```shell script
yarn lint
```

```shell script
yarn format
```

## Testing

Provided by [Jest](https://jestjs.io/).

```shell script
yarn test
```

```shell script
yarn storybook
```
